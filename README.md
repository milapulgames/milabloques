# milabloques
Repositorio oficial de la interfaz de programación por bloques "milabloques"

## Inicio rápido

Para crear una aplicación con milabloques se debe definir una **oficina**.
La oficina incluye toda la información relevante al proyecto que se está desarrollando. En particular,
 - El lenguaje (o los lenguajes) de programación.
 - El escritorio (o los escritorios) de trabajo.
 - Los idiomas disponibles.
 - Los temas disponibles.
 - Los catálogos disponibles.

Cada **lenguaje** de oficina describe la sintaxis del lenguaje de programación usado. Estos se definen a través de pequescript.

Cada **escritorio** de oficina describe un área de trabajo (el equivalente al _workspace_ de Blockly). Suele incluir una **paleta** (el equivalente al _toolbox_ de Blockly) que tiene asociado un **catálogo** (la definición de la jerarquía de categorías que se muestra en la paleta).
